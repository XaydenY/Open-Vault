# OpenVault - Enhanced Password Manager


A secure Chrome extension for managing passwords with end-to-end encryption, autofill, autosave, and a modern, user-friendly interface.

## Installation
1. Download or clone this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable Developer Mode.
4. Click "Load Unpacked" and select the `passwordmanager` folder.

## Usage
- Click the extension icon to set up your master password and add credentials.
- Autofill and autosave work automatically on supported login forms.
- Use the password generator and security dashboard for strong, safe passwords.

All data is encrypted locally and never leaves your device.
