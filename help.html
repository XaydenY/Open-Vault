OpenVault - Help & Security Guide

Welcome to OpenVault!

Your passwords are protected with military-grade encryption:

  • PBKDF2 Key Derivation: 400,000 iterations with SHA-256
  • AES-256-GCM Encryption: Authenticated encryption with integrity
  • HMAC-SHA256: Additional integrity verification
  • Local Storage Only: No data ever leaves your device
Getting Started
  1. Create a strong master password (remember it!)
  2. Add your credentials using the vault interface
  3. Enable autofill and autosave in settings
  4. Use the password generator for strong passwords
Features
  • Autofill: Automatically fills login forms on websites
  • Autosave: Saves new credentials when you log in
  • Password Generator: Creates strong, random passwords
  • Export/Import: Backup and restore your vault
  • Audit Log: Track all vault activities
Security Best Practices
  • Use a unique, strong master password
  • Enable auto-lock for security
  • Regularly export backups
  • Use generated passwords for maximum security
  • Never share your master password
Important: Your master password cannot be recovered. Keep it safe!
